# Deployment notes - DipMapper.com v19.3.0

This package is a static GitHub Pages deployment candidate. It includes `index.html`, legal pages, assets, manifest, service worker, scripts, docs, `.github/workflows/deploy.yml`, and `CNAME` for `dipmapper.com`.

## Size posture

The package is deliberately small. It stays far below GitHub Pages published-site and repository size guidance.

## Key checks before launch

1. Test from HTTPS, not only from a local file preview.
2. Confirm browser location permission appears.
3. Confirm vertical scrolling works on phone and desktop.
4. Click ranking cards while search continues.
5. Click numbered map dots while search continues.
6. Open privacy, cookies, disclaimer, imprint and advertising pages.
