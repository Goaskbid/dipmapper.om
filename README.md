# DipMapper.com v20.0.0

Mobile-first swim-place finder for nearby beaches, badis, pools, spas and open-water places.

## What is new in v20.0.0

- Mobile-first layout with sticky map, scrollable results and full-screen travel cards on phones.
- Corrected Meilen local pack: Strandbad Dorfmeilen, Strandbad Feldmeilen, Badeanlage Laendeli, Hallenbad Allmend Meilen, and Steinrad separated as Herrliberg.
- Robust result and numbered-map-dot opening while search continues.
- Stronger photo blocklist for non-swim imagery such as mosques, minarets, churches, maps, documents and generic panoramas.
- GitHub Pages deployable static package with CNAME for dipmapper.com.
